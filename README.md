# 📊 SQL Sales Data Exploration Project

## 🚀 Overview
This project focuses on exploring and analyzing a sales dataset using SQL to extract meaningful business insights.

The main objective is to transform raw data into actionable insights by applying structured queries, aggregations, and business-driven analysis.

---

## 📂 Dataset Description
The dataset consists of three main tables:

- **fact_sales** → Contains transactional sales data  
- **dim_customers** → Contains customer information  
- **dim_products** → Contains product details  

---

## 🔍 Project Workflow

### 1. Database Exploration
- Explored database structure using `INFORMATION_SCHEMA`
- Identified tables and columns

### 2. Dimensions Exploration
- Analyzed customer and product attributes
- Explored:
  - Countries
  - Categories
  - Subcategories

---

### 3. Date Analysis
- Identified:
  - First and last order dates  
  - Business activity duration  
- Analyzed customer age distribution

---

### 4. KPI & Metrics
Calculated key business metrics:

- 💰 Total Sales  
- 📦 Total Quantity  
- 💵 Average Price  
- 🧾 Total Orders  
- 👥 Total Customers  
- 🛍️ Total Products  

✔ Built a unified KPI report using `UNION ALL`

---

### 5. Magnitude Analysis
- Customer distribution by country & gender  
- Product distribution by category  
- Revenue contribution by category  
- Sales distribution across countries  

---

### 6. Ranking Analysis
- 🥇 Top 5 subcategories by revenue  
- 📉 Bottom 5 products by revenue  
- 👑 Top 10 customers by revenue  
- ⚠️ Customers with lowest number of orders  

---

## 🛠️ Tools & Technologies

- Identified top-performing product categories driving revenue
